Build it better with JavaScript

This project shows managed dependencies and build automation via JavaScript with the tools Bower and Grunt respectively.

Bower will:

  • Manage our dependencies and versions for third-party tools (ie. jquery, angular)
    • Don't put version numbers in your code! (ie. script tags) Let bower handle versions for you!!

Grunt will:

  • Automate things like:
    • compile less to css
    • minify css
    • minify html
    • lint/hint, minify, and uglify js
    • optimize images (smush 'em by removing metadata)
    • ftp files to server
    • whatever you want it to do...
    • It's amazing!!!
